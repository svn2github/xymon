NAME

clientlaunch.cfg - Task definitions for the hobbitlaunch utility

 

SYNOPSIS

~hobbit/client/etc/clientlaunch.cfg

 

DESCRIPTION

The clientlaunch.cfg file holds the list of tasks that hobbitlaunch runs on a Hobbit client. This is typically just the Hobbit client itself, but you can add custom test scripts here and have them executed regularly by the Hobbit scheduler.

 

FILE FORMAT

See the hobbitlaunch.cfg(5) description.
